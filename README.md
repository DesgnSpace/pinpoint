# Pinpoint

Load any image and click to get exact X/Y pixel coordinates and percentage positions. Instantly see image dimensions, aspect ratio, and EXIF metadata (camera, date taken, aperture, exposure, ISO, focal length).

- Drag & drop, paste from clipboard, or load via URL
- Copy values from the sidebar; reset to analyze another image
- Great for building UIs with image overlays—know exactly where to place elements and which percentage values to use
